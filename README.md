Um bot que você pode pegar o código e ageitar do seu jeito e você poderá usar na sua botlist!<br>

# Funções
- Addbot
- Firebase
- Votos
- Informações do bot
- Remover o bot automaticamente ao dono sair do servidor
- Rank de votos

E também não se esqueça de configurar o .env!
